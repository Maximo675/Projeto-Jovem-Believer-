```
Terminal 1: python run.py (deixar aberto)
Terminal 2: Outros comandos
```

### 2️⃣ Limpar cache do navegador frequentemente
```
CTRL + SHIFT + DEL (Windows/Linux)
CMD + SHIFT + DEL (Mac)
```

### 3️⃣ Consultar documentação antes de desenvolver
```
Procure em docs/ primeiro
Se não encontrar, consulte ISSUES.md
```

### 4️⃣ Fazer backups regularmente
```
Especialmente: requirements.txt, schema.sql, .env
```

```
Garante que o backend funciona
Depois integra no frontend com segurança
```
